# Atlas

## Project Overview

Social media app base on Twitter.

#### Featurew (Current Status):

- [x] Landing page
- [x] Authentication
  - [x] Sign-In Form
- [x] Home page
  - [x] Add posts
  - [x] Like post
  - [x] Side panel
- [x] Profile page
  - [x] Display personal post
  - [x] Display following
- [x] Post page
  - [x] Reply to post
- [x] Account page

  - [x] Update username
  - [x] Update picture

- [ ] Right panel
  - [ ] Search bar
  - [ ] context menu
